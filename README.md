# REST-API-using-FastAPI
I built a REST API using FastAPI with CRUD operations for employee data. I used Pydantic for request validation and FastAPI’s built‑in Swagger UI for testing. The project demonstrates my backend development and API design skills


# FastAPI REST API – Employee Management

## Description
A simple REST API built using FastAPI that performs CRUD operations on employee data.

## Features
- Create employee
- Retrieve employee(s)
- Update employee details
- Delete employee
- Swagger UI support

## Tech Stack
- Python
- FastAPI
- Uvicorn

## How to Run
```bash
pip install -r requirements.txt
uvicorn main:app --reload
